Snippets
===

使用 [jupyter](https://jupyter.readthedocs.io/en/latest/index.html) 来收集代码。

使用的版本为 python3.5+


用法
---

在根目录下使用下面命令即可

```
jupyter notebook
```

目录
---

文件 | 说明
--- | ---
sample | 一些简单的代码片段
design-patterns | 使用 python 实现的设计模式
collections | 标准库 collections 的实例
heapq | 标准库 heapq 的实例
itertools | 标准库 itertools 的实例
decorator | 装饰器相关例子
iterable | 可迭代对象的简单说明
descriptor | 装饰器的简单例子
