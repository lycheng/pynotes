Snippets
===

使用 [jupyter](https://jupyter.readthedocs.io/en/latest/index.html) 来收集代码。

使用的版本为 python3.5+


用法
---

在根目录下使用下面命令即可

```
jupyter notebook
```

目录
---

 - collections - 标准库 collections 的实例
 - data-structure - 使用 python 实现的数据结构
 - decorator - 装饰器相关例子
 - descriptor - 描述符简单例子
 - design-patterns - 使用 python 实现的设计模式
 - heapq - 标准库 heapq 的实例
 - iterable - 可迭代对象的简单说明
 - itertools - 标准库 itertools 的实例
 - sample - 一些简单的代码片段
 - tools
   - mimtproxy - mimtproxy 的简单使用
